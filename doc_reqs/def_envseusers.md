# ユーザと使用環境の定義

## ユーザ
### 基本情報
- 人間、ヒト
- 一般人
- PC操作可
- 何かしらのインターネットサービスを使い慣れている
- 実家から離れて暮らす（あるいは、自宅以外に自分の場所を持つ）
- ガジェットオタク
### サービスに求めること
- 私物の場所を簡単に把握できる
- 私物の貸し借りを管理できる
- マニュアルを必要としない
## 環境
### 自宅・実家・拠点（仕事場、学校など）
- インターネット回線が使える
- ものを配置するための十分な（１か所以上の）収納場所がある
## ことばの定義
- ガジェット
	- 任意のモノ
- 配置
	- ある場所を定位置とする
